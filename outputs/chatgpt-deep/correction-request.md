# chatgpt-deep 修正依頼

以下の点を修正し、標準モジュールに貼り付けられるVBAコード全文を再提示してください。コード内容を省略せず、修正理由も簡潔に説明してください。

1. 0件のセルは、期待結果に合わせて空欄にしてください。`DETAIL` の月合計、`SUBTOTAL` の日別列、`SUBTOTAL` の月合計は、合計が0の場合は `0` ではなく空欄にしてください。発生がある日付・行だけ数値を出してください。
2. 再実行時の事前クリアは、仕様どおり `DETAIL` 行の `H:AL` の日別転記セルを中心にしてください。`SUBTOTAL` 行と `AM` 列は最後に再計算または復元する位置づけにし、不要に広い初期クリアを避けてください。
3. `マスタ` シートの読み取り範囲を、サンプルレイアウトに合わせて区画ごとに分けてください。支店マスタは2行目見出しから次の空行または業務マスタ手前まで、業務マスタは12行目見出しから次の空行または支店・業務対応表手前まで、支店・業務対応表は24行目見出し以降を読む形にしてください。支店コード集合に業務コードが混ざる、業務コード集合に支店コードが混ざる、という状態を避けてください。
4. 日次ファイル探索では、対象を `クレーム集計YYMMDD.xlsx` に限定してください。`.xlsm`, `.xlsb`, `.xls` などは仕様上の対象外なので、通常の異常ログに混ぜないでください。
5. 28日/30日の月外日列は空欄のままにし、月合計にも小計にも含めないことを、修正版でも維持してください。

追加参照設定を使う場合は、参照設定名、理由、設定手順、late bindingまたは標準VBAだけの代替案も説明してください。
